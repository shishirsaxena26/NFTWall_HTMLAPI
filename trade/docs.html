/getManualSettings Bearer Token { "pair": "BTCUSDT" } { "data": {...} } Get user manual settings
/SaveSettings Bearer Token { "pair": "BTCUSDT", "setting": { "mode": "HYBRID", "spotConfig": {...}, "futuresConfig": {...} } } { "message": "Settings Saved" } Save/update trading settings
/activateBot Bearer Token { "pair": "BTCUSDT" } { "message": "ActivateBot Saved" } Activate bot with settings
/getRunningBots Bearer Token None { "data": [...] } List all bots with status
/startBot Bearer Token { "pair": "BTCUSDT" } { "message": "Bot started successfully" } Start bot
/stopBot Bearer Token { "pair": "BTCUSDT" } { "message": "Bot stopped successfully" } Stop bot
/getNotifications Bearer Token None { "data": [...] } Get notifications
/readNotification Bearer Token { "notificationId": 1 } { "message": "Notification marked as read" } Mark notification read
/deleteNotification Bearer Token { "notificationId": 1 } { "message": "Notification deleted successfully" } Delete notification
/logout Bearer Token None { "message": "Logout successful" } Invalidate token

🔐 Authentication Rules

  • Signup/Login: No token required
  • All other APIs: Require Authorization: Bearer TOKEN
  • API Bind: Requires OTP + Token
  • Token expires in: 1 hour

⚠️ Important Notes

  • Headers must include: Content-Type: application/json
  • Case-sensitive APIs: /SaveSettings (NOT /savesettings)
  • Token is stored server-side and revoked on logout
  • OTP is single-use
